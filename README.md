## Overview
**YellowKey BitLocker** is a powerful tool designed to bypass BitLocker encryption on Windows systems using physical access.

It exploits weaknesses in the Windows Recovery Environment (WinRE) and default TPM-only configurations.

> [!Warning]
> This tool is intended for **educational and security research purposes only**.

## Key Features
- Fast BitLocker bypass with physical USB access
- Works on TPM-only protected drives
- WinRE based attack vector
- Supports Windows 10 & Windows 11
- Simple and easy to use
- No need for recovery key in many cases
